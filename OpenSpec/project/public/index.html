Checklist

手动验证

  1. 先登录，再发送知识库内问题。
  2. 观察 planner 是否输出 evidence policy。
  3. 观察回答结束后是否展示 claims、decision 与 notes。
  4. 提问超范围问题，确认系统会保守回答或标记证据不足。
